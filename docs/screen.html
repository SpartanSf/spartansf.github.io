Screen API Documentation

Basic Functions

draw()

Draws the current contents of the graphics buffer to the screen. No arguments required.

getSize()

Returns two values: the width and height of the screen, in pixels.

Color & Transparency

setColor(R, G, B)

Sets the default drawing color. All future drawing operations will use this RGB color, unless a draw function overrides it using extra color parameters (e.g. drawPixel(x, y, R, G, B)).

setColor(Alpha)

Sets global transparency, from 1 (fully invisible) to 256 (fully opaque). Unlike RGB, alpha cannot be overridden per draw function.

setColor(R, G, B, A)

Sets both RGB color and transparency in a single call. Useful for defining full appearance in one go.

Rotation

setRotation(Angle, X, Y)

Sets a rotation transform for all upcoming complex drawings. Rotates around the point (X, Y) by Angle degrees.

setRotation(Angle)

Rotates only the next complex drawing operation by Angle degrees, centered on its own geometry.

setRotation()

Clears any previously set rotation transformations.

Simple Drawing

drawPixel(X, Y)

Draws a single pixel at coordinate (X, Y).

drawLine(x1, y1, x2, y2)

Draws a straight line from (x1, y1) to (x2, y2).

fill(x1, y1, x2, y2)

Fills the rectangular region defined by (x1, y1) and (x2, y2) with the current color.

fill(R, G, B)

Clears the screen using the specified RGB color. This is commonly used for full-screen color fills and can be combined with transparency for fade transitions.

floodFill(X, Y, R, G, B)

Performs a strict flood fill from (X, Y), replacing all adjacent pixels with the exact starting color.

floodFill(X, Y, Tolerance, R, G, B)

Similar to the above, but allows a color difference defined by Tolerance to match similar colors.

Complex Drawing

drawRectangle(x1, y1, x2, y2)

Draws a wireframe rectangle around the bounding box defined by (x1, y1) and (x2, y2).

drawCircle(x1, y1, x2, y2)

Draws a circle or ellipse constrained within the bounding box defined by (x1, y1) and (x2, y2).

drawCircle(X, Y, Radius)

Draws a circle centered at (X, Y) with the specified radius.

drawPolygon(X, Y, n, Radius)

Draws a regular polygon centered at (X, Y) with n sides, fitting within the given radius.

toRadius(n, Apothem)

Helper function to convert an apothem length into a radius for n-sided polygons. Useful for sizing with known apothem.

drawBezier(x1, y1, x2, y2, x3, y3)

Draws a smooth BÃ©zier curve through the provided control points.

drawSpline(Xs, Ys)

Draws a spline curve through the provided X and Y value arrays.

drawSpline(Coordinates)

Draws a spline from a flat list of interleaved X and Y coordinates (e.g. {x1, y1, x2, y2, ...}).
