Screen API Documentation

Basic Functions

draw()

Draws the current contents of the graphics buffer to the screen.
Parameters: None
Returns: void

getSize()

Returns the width and height of the screen in pixels.
Parameters: None
Returns: int, int – width, height
Example: local w, h = getSize()

Color & Transparency

setColor(R, G, B)

Sets the default RGB drawing color.

Parameters:
  • R: int (0–255)
  • G: int (0–255)
  • B: int (0–255)
Returns: void

setColor(Alpha)

Sets the global transparency level.

Parameters:
  • Alpha: int 1–256
Returns: void

setColor(R, G, B, A)

Sets both color and transparency.

Parameters:
  • R: int
  • G: int
  • B: int
  • A: int 1–256
Returns: void

Rotation

setRotation(Angle, X, Y)

Applies rotation around a specific point.

Parameters:
  • Angle: float (degrees)
  • X: int
  • Y: int
Returns: void
Example: setRotation(45, 100, 100)

setRotation(Angle)

Applies rotation to the next complex draw operation.

Parameters:
  • Angle: float
Returns: void

setRotation()

Clears any active rotation.
Parameters: None
Returns: void

Simple Drawing

drawPixel(X, Y)

Draws a single pixel at coordinate (X, Y).

Parameters:
  • X: int
  • Y: int
Returns: void
Example: drawPixel(100, 50)

drawLine(x1, y1, x2, y2)

Draws a straight line from x1, y1 to x2, y2.

Parameters:
  • x1: int – Start X
  • y1: int – Start Y
  • x2: int – End X
  • y2: int – End Y
Returns: void
Example: drawLine(10, 10, 200, 200)

fill(x1, y1, x2, y2)

Fills the rectangular region defined by two corners with the current drawing color.

Parameters:
  • x1: int
  • y1: int
  • x2: int
  • y2: int
Returns: void
Example: fill(50, 50, 150, 100)

fill(R, G, B)

Clears the entire screen using the specified RGB color. Often used for background fills or fades.

Parameters:
  • R: int 0–255
  • G: int 0–255
  • B: int 0–255
Returns: void
Example: fill(0, 0, 0)

floodFill(X, Y, R, G, B)

Performs a strict flood fill starting at X, Y. Only replaces pixels matching the starting color exactly.

Parameters:
  • X: int
  • Y: int
  • R: int 0–255
  • G: int 0–255
  • B: int 0–255
Returns: void
Example: floodFill(100, 100, 255, 0, 0)

floodFill(X, Y, Tolerance, R, G, B)

Performs a flood fill with a tolerance. Pixels that are similar in color (within Tolerance) are replaced.

Parameters:
  • X: int
  • Y: int
  • Tolerance: int 0–255
  • R: int 0–255
  • G: int 0–255
  • B: int 0–255
Returns: void
Example: floodFill(100, 100, 10, 0, 255, 0)

Complex Drawing

drawPolygon(X, Y, n, Radius)

Draws a regular polygon centered at X, Y.

Parameters:
  • X: int
  • Y: int
  • n: int – Number of sides
  • Radius: float – Distance from center to a vertex
Returns: void
Example: drawPolygon(150, 150, 6, 40)

toRadius(n, Apothem)

Converts an apothem to a radius.

Parameters:
  • n: int – Number of polygon sides
  • Apothem: float
Returns: float – Calculated radius
Example: local r = toRadius(6, 50)

drawBezier(x1, y1, x2, y2, x3, y3)

Draws a quadratic Bézier curve using 3 control points.

Parameters:
  • x1, y1: int – Start point
  • x2, y2: int – Control point
  • x3, y3: int – End point
Returns: void
Example: drawBezier(10, 100, 60, 20, 110, 100)

drawSpline(Xs, Ys)

Draws a spline curve using parallel arrays.

Parameters:
  • Xs: array of int – X coordinates
  • Ys: array of int – Y coordinates
Returns: void
Example: drawSpline({10, 30, 60}, {90, 20, 90})

drawSpline(Coordinates)

Draws a spline from interleaved coordinate array.

Parameters:
  • Coordinates: array of int – {x1, y1, x2, y2, ...}
Returns: void
Example: drawSpline({10, 90, 30, 20, 60, 90});
